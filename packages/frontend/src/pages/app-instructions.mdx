The BPM App allows participant to:

- See the conference agenda, and build the personalized schedule
- See the list of people attending the conference
- Share messages with the other attendees
- Share pictures with the other attendees

If you want to try it, you can go to https://www.bpm2025seville.org/app/ (from your mobile device) and register with the same email you used to register for the conference.

# Installation instructions for Android

The default Chrome browser is capable of installing the ICPM App. First, navigate to https://www.bpm2025seville.org/app/ using Chrome. Press the "three dot” icon in the upper right to open the menu. Select "Add to Home screen.” Press the "Add” button in the popup. The BPM App is now installed and available on your home screen.

![Installation instructions for Android](/program/image-android.png)

# Installation instructions for iOS

BPM App can only be installed on iOS using the Safari browser. First, navigate to https://www.bpm2025seville.org/app/ using Safari. Press the "Share” button, scroll the popup menu down and select "Add to Home Screen”. Lastly, tap "Add” in the top right corner to finish the installation. It will now be on your home screen.

![Installation instructions for iOS](/program/image-iphone.png)
